# vi_Turkce_Dokuman
* ozellikle Linux'da 'vi' kullanimini yayginlastirmak icin hazirlanmis bir diger Turkce kaynak

Komut Mode
----------
```python
<Esc>   # Komut Mode'a geç
<:x!>   # Kaydet ve çık
<:wq!>  # Kaydet ve çık 
<1G>    # Dosya başı, ilk satır
<G>     # Dosya sonu, en son satır
```

Editor Mode
-----------
```python
<dd>  # cursorun bulundugu satiri siler
```
